📌 Overview

The Hospital Appointment Booking System is a web-based application designed to simplify booking and managing medical appointments. Patients can schedule appointments with doctors, while administrators manage doctors, schedules, and bookings.

✨ Features
👤 Patient
Register and login
View available doctors
Book appointments
View appointment history
🛠️ Admin
Add, update, and delete doctors
Manage doctor availability
View all appointments
Manage patient records
💻 Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Hosting: InfinityFree
🗄️ Database Structure (Basic)
patients
doctors
appointments
🚀 How to Use (Live Site)
Open the live link
Register as a patient or login as admin
Book and manage appointments
